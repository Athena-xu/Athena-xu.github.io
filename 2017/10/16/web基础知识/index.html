web发展史

w3>>Web发展史：Mosaic 1993年–Spy Glass公司–Netscape(网景)–Microsoft(比尔盖茨)–IE；
JavaScript–Netscape Navigator浏览器第一次出现1996年，作者Brendan Eich;
浏览器的基本组成部分：外核、内核 （Rendering Engine）；
内核：1.渲染引擎（处理HTML+CSS）；2.JS引擎(处理JS)；3.其他部分
// 浏览器快慢大部分指得是JavaScript引擎的渲染速度（跑分测试）；

2001年发布ie6,首次实现对js引擎的优化和分离（可以跑上万行代码）；
2008年Google发布最新浏览器Chrome，采用优化后的JS引擎，代号V8,能把JS代码直接转化为机械码（01010101…），速度快;chrome是运行、渲染最快的浏览器；
后来Firefox也推出了强大功能的JS引擎，Firefox3.5 TraceMonkey(对频繁执行的代码做了路径优化)；Firefox4.0 JeagerMonkey引擎;

常见主流浏览器内核：
Chrome—-Webkit(和Apple共同开发)->Blink； JS引擎—V8；
Mozilla Firefox—-Gecko； JS引擎— SpiderMonkey（1.0-3.0）/ TraceMonkey（3.5-3.6）/ JaegerMonkey（4.0-）；
IE/Edge(win10)—-Trident； JS引擎—EdgeJScript（IE3.0-IE8.0） / Chakra（IE9+之后，查克拉)；
Safari—-Webkit； JS引擎— Nitro（4-）;
Opera—-Presto->Blink； JS引擎— Linear A（4.0-6.1）/ Linear B（7.0-9.2）/ Futhark（9.5-10.2）/ Carakan（10.5-）；
